== Setting Up the Environment ==

Need iinstall openssl library, (for cmake need add variables OPENSSL_ROOT_DIR to bash)

== Build and Run ==

Just make build folder, run CMake and then edit/build the project:

```
md .build
cd .build
cmake ..
cmake --build .
```
